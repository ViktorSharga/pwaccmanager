1. Overview

This guide provides complete implementation instructions for an automated game server selection system. The solution detects when a game client shows a server selection screen and automatically selects the desired server.

Features

  • Automatic Detection: Detects server selection screen appearance
  • Server Selection: Automatically clicks on specified server
  • Confirmation: Handles confirmation button clicks
  • Multiple Approaches: Coordinate-based and image recognition methods
  • Error Handling: Robust error handling and logging
  • Calibration Tools: Built-in coordinate calibration system

Supported Methods

  1. Simple Coordinate Method - Easy setup, coordinate-based clicking
  2. Advanced Image Recognition - OpenCV-based template matching
  3. Full Auto-Launcher - Complete game launch and server selection

4. Configuration

Method 1: Simple Coordinate Configuration

Step 1: Coordinate Calibration

  1. Launch your game manually
  2. Navigate to server selection screen
  3. Run calibration script:
python 
# calibration.py
import pyautogui
import time

def calibrate_coordinates():
    print("🎯 COORDINATE CALIBRATION")
    print("Move mouse over Asgard server and press Ctrl+C")
    
    try:
        while True:
            x, y = pyautogui.position()
            print(f"\rMouse: ({x}, {y}) ", end="")
            time.sleep(0.1)
    except KeyboardInterrupt:
        asgard_coords = (x, y)
        print(f"\n✅ Asgard: {asgard_coords}")
    
    print("\nMove mouse over SELECT button and press Ctrl+C")
    try:
        while True:
            x, y = pyautogui.position()
            print(f"\rMouse: ({x}, {y}) ", end="")
            time.sleep(0.1)
    except KeyboardInterrupt:
        select_coords = (x, y)
        print(f"\n✅ Select: {select_coords}")
    
    print("\n📋 Configuration:")
    print(f"ASGARD_COORDS = {asgard_coords}")
    print(f"SELECT_COORDS = {select_coords}")

if __name__ == "__main__":
    calibrate_coordinates()

5. Implementation Steps

Quick Start Implementation

Step 1: Create Main Script

Save as server_selector.py:

python 
import pyautogui
import time
from PIL import ImageGrab
import logging

class GameServerSelector:
    def __init__(self):
        # Configure logging
        logging.basicConfig(
            level=logging.INFO,
            format='%(asctime)s - %(levelname)s - %(message)s',
            handlers=[
                logging.FileHandler('server_selector.log'),
                logging.StreamHandler()
            ]
        )
        self.logger = logging.getLogger(__name__)
        
        # Configuration
        pyautogui.FAILSAFE = True  # Move mouse to corner to stop
        pyautogui.PAUSE = 0.5      # Pause between actions
        
        # Coordinates (UPDATE FROM CALIBRATION)
        self.asgard_coords = (447, 390)
        self.select_coords = (642, 451)
    
    def detect_server_screen(self, timeout=30):
        """Detect server selection screen"""
        self.logger.info("🔍 Waiting for server selection screen...")
        start_time = time.time()
        
        while time.time() - start_time < timeout:
            try:
                screenshot = ImageGrab.grab()
                
                # Check for UI colors typical of server selection
                check_points = [
                    (400, 300), (500, 350), (600, 400),
                    (350, 380), (450, 420)
                ]
                
                brown_ui_count = 0
                for x, y in check_points:
                    try:
                        r, g, b = screenshot.getpixel((x, y))[:3]
                        # Brown UI detection
                        if 80 < r < 180 and 60 < g < 140 and 30 < b < 90:
                            brown_ui_count += 1
                    except:
                        continue
                
                if brown_ui_count >= 3:
                    self.logger.info("✅ Server selection screen detected!")
                    return True
                    
            except Exception as e:
                self.logger.error(f"Detection error: {e}")
            
            time.sleep(1)
        
        self.logger.warning("❌ Server screen not detected within timeout")
        return False
    
    def select_server(self):
        """Select Asgard server and confirm"""
        try:
            self.logger.info("🖱️ Selecting Asgard server...")
            
            # Click Asgard server
            pyautogui.click(self.asgard_coords[0], self.asgard_coords[1])
            time.sleep(1.5)
            
            # Click Select button
            self.logger.info("✅ Clicking Select button...")
            pyautogui.click(self.select_coords[0], self.select_coords[1])
            time.sleep(1)
            
            self.logger.info("🎉 Server selection completed!")
            return True
            
        except Exception as e:
            self.logger.error(f"❌ Selection error: {e}")
            return False
    
    def run_auto_selection(self):
        """Main workflow"""
        self.logger.info("🚀 Starting automatic server selection...")
        
        # Wait for server screen
        if not self.detect_server_screen(timeout=60):
            return False
        
        # Wait for UI to stabilize
        time.sleep(2)
        
        # Select server
        return self.select_server()

def main():
    selector = GameServerSelector()
    
    print("🎮 Game Server Auto-Selector")
    print("📋 Instructions:")
    print("1. Launch your game")
    print("2. Wait for server selection screen")
    print("3. This script will automatically select Asgard")
    print("\n⚠️  Move mouse to top-left corner to emergency stop")
    print("\n🚀 Starting in 5 seconds...")
    
    time.sleep(5)
    
    success = selector.run_auto_selection()
    
    if success:
        print("\n✅ SUCCESS: Server selected automatically!")
    else:
        print("\n❌ FAILED: Manual intervention required")
        print("💡 Check coordinates and screen detection settings")

if __name__ == "__main__":
    main()

6. Troubleshooting

Common Issues

Issue 1: Python Not Found

Error: 'python' is not recognized as an internal or external command

Solution:

  1. Reinstall Python with "Add to PATH" checked
  2. Or manually add Python to PATH:
    • Open System Properties → Environment Variables
    • Add Python installation directory to PATH
    • Restart Command Prompt

Issue 2: Permission Denied

Error: Access denied or automation blocked

Solution:

  1. Run Command Prompt as Administrator
  2. Disable antivirus temporarily during setup
  3. Add Python to antivirus exceptions

Issue 3: Coordinates Not Working

Error: Clicking wrong locations

Solution:

  1. Re-run coordinate calibration
  2. Check screen resolution consistency
  3. Ensure game is in windowed mode (not fullscreen)
  4. Verify DPI scaling settings

Debug Mode

Enable detailed debugging by adding to your script:

import logging
logging.basicConfig(level=logging.DEBUG)

SERVERS = {

    return False
